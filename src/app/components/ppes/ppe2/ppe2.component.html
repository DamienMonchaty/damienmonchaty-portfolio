FormulaNews - Contexte

Le site d’actualité de sport automobile FormulaNews, desirant gagner en visibilité et en audience décide de demander au service développement web de concevoir une application mobile diffusant les résultats de toutes les courses de Formule 1 depuis 1950. Elle permettra également d’avoir des informations sur le championnat en cours en termes de pilotes, constructeurs & classement

Technologies utilisées :

  • Xamarin Forms
  • SQLite
  • Angular
  • Git

Contraintes diverses :

  • Respect des ressources numériques

Ressources utilisées :

  • Microsoft Visual Studio Community 2022
  • Visual Studio Code
  • GitHub
  • Trello

Quelques fonctionnalités :

  • Information sur la prochaine course & dernière course
  • Mise en place d’un blog
  • Consultation des résultats de toutes les courses, pilotes, contructeurs, courses de Formule 1 depuis 1950
  • Une utilisation facile & intuitive de l’application
