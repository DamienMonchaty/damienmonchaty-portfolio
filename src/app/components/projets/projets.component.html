Glpi / Wamp Server
Télécharger le projet

Présentation

Git : https://github.com/DamienMonchaty/wampglpi

GLPI - Gestionnaire Libre de Parc Informatique est un logiciel libre de gestion des services informatiques et de gestion des services d’assistance.
Cette solution libre est éditée en PHP et distribuée sous licence GPL.
GLPI est une application web qui aide les entreprises à gérer leur système d’information.
Il est compatible avec toutes les versions d’OS soit Linux/Windows ainsi que MAC OS avec l’aide d’une machine virtuelle.
Pour utiliser cet outil, il faut :
  • WampServer avec Apache pour la gestion de la base de donnée
  • Une version de GLPI

MyCellar est une REST API de recettes de cuisine.
Elle permet selon une selection de produits de consulter une suggestion de recettes
L'application répertorie des informations concernant les produits alimentaires comme :
  • Ingrédients
  • Catégories
  • Informations nutritionelles
